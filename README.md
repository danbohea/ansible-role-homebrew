# Ansible role: Homebrew

Installs Homebrew & Homebrew Cask on Mac OS X.

## Requirements

- Mac OS X


## Role Variables

None.


## Dependencies

None.


## Example Playbook

```
- hosts: macbookair
  connection: local

  roles:
    - role: ansible-role-homebrew
```
